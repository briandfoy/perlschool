Perl New Features

By brian d foy

Publication date: 06 Jan 2021

In 2010, Joshua McAdams and brian d foy updated for v5.10 Joseph Hall's seminal book Effective Perl Programming. At the same time, they set up a website (https://www.effectiveperlprogramming.com) to continue that work, and in the next 10 years they posted hundreds more articles, many of which explained new features that showed up after v5.10. brian has collected and updated the items explicating those new features.
